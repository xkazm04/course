## Context Information

**Context Name**: curriculum-generator - AI Content Generation

**Context Description**:
## Overview

The curriculum-generator feature provides an LLM-powered system for generating personalized curriculum content. It creates lesson outlines, code exercises, quizzes, and project specifications tailored to the user's learning profile and career goals, integrated with the Career Oracle learning path system.

## Key Capabilities

- **Personalized Content Generation**: Creates lessons, exercises, quizzes, and projects customized to user's skill level, target role, and learning style
- **Smart Caching System**: Caches generated content with 7-day expiry, hit tracking, and quality metrics to minimize API calls and enable community curation
- **Feedback Loop Integration**: Collects user ratings on clarity, difficulty, relevance, and engagement to continuously improve content quality
- **Career Oracle Integration**: Seamlessly generates curriculum when learning paths are created, with preloading support for upcoming modules

## Architecture

### Component Breakdown
| Component/File | Purpose | Layer |
|----------------|---------|-------|
| `lib/types.ts` | Comprehensive TypeScript types for lessons, exercises, quizzes, projects, feedback, and cache | Data |
| `lib/promptBuilder.ts` | Builds structured prompts for LLM curriculum generation | Service |
| `lib/curriculumGenerator.ts` | Core service for generating and assembling curriculum content via API | Service |
| `lib/curriculumStorage.ts` | LocalStorage management for caching, feedback, completion tracking, and quality metrics | Data |
| `lib/useCurriculum.ts` | React hook for curriculum generation, progress tracking, and content access | Hook |
| `lib/useCareerOracleCurriculum.ts` | Hook integrating curriculum with Career Oracle path system | Hook |
| `components/LessonViewer.tsx` | Displays lessons with expandable sections, code snippets, tips, and concepts | UI |
| `components/CurriculumOverview.tsx` | Shows curriculum progress with stats, content list, and skills covered | UI |
| `components/FeedbackModal.tsx` | Multi-step modal for collecting detailed user feedback on content | UI |
| `components/index.ts` | Component exports barrel file | Export |
| `index.ts` | Feature exports for types, functions, hooks, and components | Export |

### Data Flow
1. User selects a learning path module in Career Oracle
2. `useCareerOracleCurriculum` builds a CurriculumGenerationRequest with module info and user profile
3. System checks cache via `curriculumStorage.getCached()`
4. If miss, `generateCurriculum()` calls API with prompts from `promptBuilder.ts`
5. Response is cached, saved to user curricula, and rendered via `CurriculumOverview`
6. User views lessons via `LessonViewer`, with completion tracked in storage
7. After completion, `FeedbackModal` collects ratings that update quality metrics

### Key Dependencies
- **External**: React, framer-motion, lucide-react icons
- **Internal**: PrismaticCard (shared/components), storageFactory (shared/lib), goal-path/lib/predictiveTypes
